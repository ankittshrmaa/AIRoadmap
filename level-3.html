LEVEL 3

Deep Learning Foundations

Master neural networks from first principles to practical implementation in PyTorch

12 weeks intensive
Goal: Implement networks from scratch

What You'll Be Able to Do at the End

  • Understand exactly how a neural network learns (forward pass → loss → backpropagation → parameter updates)
  • Implement fully-connected networks, CNNs, RNNs/LSTMs without "magic" libraries doing it for you
  • Train models in PyTorch for image, text, and sequence tasks
  • Use GPU acceleration and avoid common training mistakes
  • Build at least 5 showcase projects that demonstrate deep learning in action

Weekly Breakdown

Neural Network Fundamentals

Weeks 1-2

Goal: Understand what's happening under the hood before touching PyTorch

Topics

  • Perceptrons & multilayer perceptrons (MLPs)
  • Activation functions: Sigmoid, Tanh, ReLU, LeakyReLU, Softmax
  • Loss functions: MSE, Cross-Entropy
  • Gradient descent variants: SGD, Momentum, Adam
  • Backpropagation math: chain rule through a computational graph
  • Weight initialization (Xavier, He)
  • Overfitting & regularization: L1/L2, dropout

Hands-on

  • Week 1: Build a 2-layer neural net from scratch in NumPy
  • Week 2: Add multiple layers, different activations, and dropout

Micro-projects

  • XOR classifier from scratch
  • Handwritten digit recognition (MNIST) with NumPy (no frameworks)

PyTorch Basics

Weeks 3-4

Goal: Move from scratch implementations to a professional DL framework

Topics

  • Tensors, gradients, and autograd in PyTorch
  • Building nn.Module classes
  • Training loop: forward → loss → backward → optimizer step
  • Dataset & DataLoader
  • GPU acceleration (.to(device) usage)
  • Saving & loading models

Hands-on

  • Rewrite your NumPy MNIST classifier in PyTorch
  • Train a model to classify CIFAR-10 images

Micro-projects

  • MNIST digit classifier in PyTorch
  • CIFAR-10 image classifier with simple CNN

Convolutional Neural Networks

Weeks 5-6

Goal: Learn image processing in deep learning

Topics

  • Convolutions, filters/kernels, feature maps
  • Pooling (max, average)
  • Padding, stride
  • CNN architectures: LeNet, AlexNet, VGG, ResNet
  • Transfer learning & fine-tuning

Hands-on

  • Implement your first CNN from scratch in PyTorch
  • Apply transfer learning using a pre-trained ResNet

Micro-projects

  • Fashion-MNIST image classifier (CNN)
  • Cat-vs-Dog classifier with ResNet fine-tuning

RNNs & LSTMs

Weeks 7-8

Goal: Handle sequential data like text and time series

Topics

  • RNN architecture & vanishing gradients
  • LSTMs & GRUs for long-term dependencies
  • Word embeddings (Word2Vec, GloVe)
  • Sequence-to-sequence (seq2seq) basics

Hands-on

  • Train an LSTM to predict next character in text
  • Build a sentiment analysis model for IMDB reviews

Micro-projects

  • Character-level text generator
  • Movie review sentiment classifier

Training Best Practices

Weeks 9-10

Goal: Learn how to train models faster, better, and without overfitting

Topics

  • Learning rate schedules
  • Data augmentation (image flips, rotations, noise)
  • Early stopping & checkpointing
  • Mixed precision training for speed
  • TensorBoard for visualization

Hands-on

  • Optimize CIFAR-10 classifier for >85% accuracy
  • Add real-time TensorBoard logging

Micro-projects

  • Image augmentation comparison experiment
  • Training log dashboard in TensorBoard

Capstone Project

Weeks 11-12

Goal: End-to-end deep learning project from dataset to deployment

Project Ideas

  • Face Mask Detection System (CNN + transfer learning)
  • Chatbot with LSTM (train on movie dialogue dataset)
  • Music Genre Classifier (audio → spectrogram → CNN)

Deliverables

  • Clean PyTorch code
  • README with model architecture and results
  • Training curves & example outputs
  • Model deployed as API (FastAPI + Docker)

Stage 3 Exit Criteria

You're ready for Stage 4 (specialization) when you can:

  • Build MLP, CNN, and LSTM from scratch in NumPy
  • Train & evaluate deep learning models in PyTorch
  • Use GPUs, checkpoints, and TensorBoard
  • Explain activation functions and backpropagation without notes
  • Have 5+ deep learning projects on GitHub with clean documentation
