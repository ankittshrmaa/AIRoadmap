LEVEL 2

Core Machine Learning

Master the fundamental algorithms and techniques that power modern AI systems

3 months
Goal: Build ML models from scratch
01

ML Fundamentals

4 weeks

Master These Concepts:

  • Supervised vs. unsupervised learning
  • Bias-variance tradeoff
  • Training/test splits and cross-validation
  • Performance metrics (accuracy, precision, recall)
  • Feature engineering techniques

Recommended Resources:

• Google's Machine Learning Crash Course
• Hands-On Machine Learning with Scikit-Learn
• Andrew Ng's ML Coursera (Weeks 1-3)
02

Core Algorithms

6 weeks

Implement These Algorithms:

  • Linear & logistic regression
  • Decision trees and random forests
  • Support Vector Machines (SVMs)
  • k-Nearest Neighbors (kNN)
  • Naive Bayes classifiers
  • Dimensionality reduction (PCA)

Recommended Resources:

• Scikit-learn documentation
• ML from Scratch (GitHub repo)
• StatQuest algorithm explanations
03

Model Optimization

5 weeks

Master These Skills:

  • Hyperparameter tuning (GridSearch, RandomSearch)
  • Regularization techniques
  • Ensemble methods (bagging, boosting)
  • Model interpretability (SHAP, LIME)
  • Basic model deployment

Recommended Resources:

• Kaggle ML courses
• Interpretable ML book
• Fast.ai Practical ML

Pro Tip

Implement each algorithm from scratch (using NumPy) before relying on scikit-learn. This deep understanding will help you tremendously when you need to debug models or customize algorithms for specific problems. Start with linear regression, then move to more complex algorithms.
